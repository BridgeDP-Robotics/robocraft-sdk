# RoboCraft SDK

> RoboCraft 是面向机器人本体厂商与应用团队的机器人动作能力开发平台，支持跨机型动作开发、训练与部署。
> RoboCraft is a motion capability development platform for robot makers and application teams, supporting cross-platform motion development, training, and deployment.

本仓库用于发布 **RoboCraft SDK 的规范文档**（接口规范、数据格式、使用说明等）。
This repository hosts the **specification documents for the RoboCraft SDK** (interface specs, data formats, usage guides).
